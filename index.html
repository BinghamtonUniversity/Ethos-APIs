Search Tips

  • Search by table or field name when an API name is not specific enough or you do not know which API to use. For example, searching for pidm will return all endpoints where that value appears in paths, descriptions, or responses, even if the API name does not include "pidm".
  • Search for spriden to find endpoints where that value appears in paths, descriptions, or responses.
  • Use quotes, such as "resources from SPRIDEN" or "resource in SPRIDEN", to narrow results to an exact phrase.
